# Apofema2D
2D редактор для создания и визуализации геометрических объектов на плоскости.

## Установка для Windows
1. Зайдите в раздел релизов.
2. Выберите понравившуюся версию программы. Начиная с версии 2.3.1 для установки понадобится скачать и распаковать у себя на компьютере архив с необходимыми файлами
#### До версии 2.3.1
1. Скачайте exe-файл.
2. Запустите exe-файл на компьютере.
#### С версии 2.3.1
1. Скачайте самый первый в списке архив выбранной версии. Это необхоимые файлы для запуска
2. Распакуйте архив на компьютере
3. Запустите файл _launch_programm.vbs_

## Использование
Сверху находятся кнопки для выбора инструмента рисования. Можно выбрать любую и с помощью зажатия ЛКМ что-нибудь нарисовать на клетчатом поле. Для удобства можно создавать подписи. Инструментом под названием "Кривая" можно провести ломанную линию, выбрав ключевые точки через которые должна пройти линия и нажать клавишу Enter. При желании в режиме проектирования можно заставить программу проанализировать линию для нахождения возможной фигуры и её сторон, углов, периметра, площади. Учтите, что не все фигуры могут быть найдены, а также то, что программа может ошибаться на 1 единицу. С помощью кисти можно проводить ограниченные в длине линии. Ластиком можно удалить объект с поля щелчком, а заливкой можно поменять цвет или закрасить фигуру.
Используйте кнопки «Очистить» для очистки поля и «Отменить измение» для удаления с поля последнего созданного объекта. Кнопкой "Смена цвета" можно быстро сменить действующий цвет на один из стандартных.
В разделе настроек можно изменить некоторые параметры приложения такие как, язык или активный цвет.
Используйте кнопки экспорта, чтобы экспортировать в программу чужую расстановку фигур или чужие настройки. Обратите внимание, что файлы обязательно должны иметь расширение json и называться как "settings" или "objects". Соблюдайте осторожность при передаче и загрузке этих файлов.

## Предназначение
Программа преимущественно расчитана для помощи в представлении геометрических фигур на плоскости. Однако можно экспериментировать и создавать уникальные объекты, используя её как средство рисования.
